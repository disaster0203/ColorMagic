Detailed Description

Static class for color calculations like mixing two of them.

This static class implements various functions for color calculations.

color_space::color_base * color_manipulation::color_calculation::add ( color_space::color_base *  color1,
color_space::color_base *  color2,
float  weight1 = 1.f,
float  weight2 = 1.f,
bool  include_alpha = false 
)
static

Static function that adds two colors.

Depending on the type of the first input color the calculation is done differently (additive or subtractive). The amount of the two colors can be varied in the range [0,1] by using their corresponding weights.

Parameters
color1The first of the two colors to add.
color2The second of the two colors to add.
weight1The first colors weight in the range [0,1].
weight2The second colors weight in the range [0,1].
include_alphaWhether alpha should be included in the calculation or not.
Returns
a new color object with the resulting color of this calculation.
