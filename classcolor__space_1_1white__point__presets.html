Detailed Description

Class that stores some default tristimulus and chromaticity coordinates.

This class stores tristimulus objects and chromaticity coordinates like A, B, C, Equal Energy, D50, or D65. The values for these tristimulus objects were taken from https://www.easyrgb.com
