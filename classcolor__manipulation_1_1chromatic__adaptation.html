Detailed Description

Static class for chromatic adaptation.

This static class implements von Kries, Bradford, XYZ Scaling and CMCCAT2000 as chromatic adaptation methods.

Transforms a given color with the given target white point by using Bradford method.

Transforms a given color with the given target white point. The color will first be converted to xyz space. Afterwards the transformation is done using Bradford matrices and finally the result is converted back to input color space.

Parameters

The transformed color in the input color space.
